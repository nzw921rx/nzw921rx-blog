我的个人博客第一版正式上线了。它不一定复杂，但对我来说是一个长期写作和持续沉淀的起点。

## 为什么搭建博客

搭建博客的初衷，是希望有一个属于自己的空间，用来系统记录技术学习、项目实践和日常思考。相比零散地把笔记放在不同平台，个人博客更适合沉淀可长期维护的内容，也方便回顾自己的成长轨迹。

博客也是一种输出倒逼输入的方式。当我尝试把一个问题讲清楚时，往往会发现自己还需要补齐哪些知识点。这个过程可以帮助我更扎实地理解技术，也能把踩过的坑、做过的取舍和解决方案记录下来。

## 为什么选择 Hugo + PaperMod

我选择 Hugo，是因为它足够轻量、生成速度快、部署简单，非常适合个人技术博客。文章可以用 Markdown 编写，内容结构清晰，也便于使用 Git 进行版本管理。

PaperMod 则提供了简洁、专注阅读的界面。它默认样式清爽，配置成本较低，同时又保留了比较灵活的自定义空间。对于第一版博客来说，我更希望先把内容发布流程跑通，而不是在复杂的主题改造上投入太多时间。

## 后续计划

接下来，我计划逐步完善这个博客：

- 补充更完整的个人介绍和项目索引
- 持续更新技术文章和项目复盘
- 优化分类、标签、归档和搜索体验
- 整理 Hugo 与 PaperMod 的配置经验
- 建立稳定的写作与发布节奏

第一版只是开始。后续我会继续把这里作为长期记录和分享的地方，让它随着我的学习和项目一起成长。
